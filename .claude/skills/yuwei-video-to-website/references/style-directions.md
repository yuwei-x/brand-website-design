# 风格方向参考

根据品牌调性选择风格方向。不同风格的字体、配色、动画策略差异极大。

## Apple 极简

- **字体**：系统字体族（SF Pro / -apple-system），标题正文同族不同字重
- **配色**：纯黑背景、白/灰文字层级、品牌蓝 `#0071e3` 仅用于 CTA
- **动画**：全部 fade-up，统一方向，克制的 30px 位移，`power2.out`
- **排版**：标题 `font-weight: 700`（不用 900），`letter-spacing: -0.02em`
- **CTA**：标准蓝色圆角按钮 + "进一步了解" 文字链接
- **特效**：通常不使用视觉特效（无霓虹、无故障、无粒子）
- **整体感觉**：克制、精确、呼吸感

## 科技炫酷

- **字体**：标题用 Display 字体（Space Grotesk / Inter 900），正文用 Body 字体（DM Sans / Inter 400），必须不同
- **配色**：从品牌色派生完整色阶，避免使用默认色板
- **动画**：4+ 种入场类型交替使用，不连续重复。详见 `animation-types.md`
- **特效**：霓虹发光、故障特效、闪烁粒子、marquee 大字等可选。详见 `visual-effects.md`
- **阴影**：多层带颜色倾向，不用扁平 `shadow-md`
- **整体感觉**：大胆、前卫、视觉冲击

## 编辑杂志风

- **字体**：标题用 Serif（Playfair Display），正文用 Sans（Source Sans 3）
- **配色**：大面积留白，极少量强调色
- **动画**：clip-reveal 为主，配合大字号排版节奏
- **排版**：超大标题 + 极细正文形成对比
- **特效**：通常不使用，偶尔用走马灯大字
- **整体感觉**：优雅、从容、编辑质感

## 配色原则（通用）

不要用默认色板偷懒。从用户给的品牌色出发，用 HSL 调整派生出色阶：
- 主色：品牌色原值
- 亮色变体：S 不变，L +15-20%
- 暗色变体：S +5%，L -20%
- 文字色：深色背景不低于 `#ccc`，浅色背景正文不浅于 `#666`

## 响应式原则（通用）

移动端（< 768px）：
- 侧对齐改居中 + 深色毛玻璃背景保证可读性
- 总滚动高度降至 ~550vh
- 字号取 clamp 下限值
- 帧数可降至 <150，宽度降至 1280px
